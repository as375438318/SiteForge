### 1.1 核心商业逻辑

> **License 买断 + 年度服务费 + 增值服务**

| 收入类型 | 模式 | 占比预估 | 特点 |
|----------|------|---------|------|
| License 买断 | 一次性，永久使用当前版本 | 60% | 核心收入，客户接受度高（"买断"比"永久订阅"有心理优势） |
| 年度服务费 | 年付，含版本升级+技术支持 | 25% | 持续性收入，续费率目标 ≥ 70% |
| 增值服务 | 按次/按月 | 15% | 托管部署/代建站/代运营，高毛利 |

**与 SaaS 的本质区别**：

| 维度 | SaaS 订阅制 | SiteForge License 买断 |
|------|------------|----------------------|
| 付费心理 | "每年都要交钱，停了就没了" | "一次买断，永久可用" |
| 客户控制权 | 低（平台停服则全丢） | 高（软件在自己服务器，永久可用） |
| 续费压力 | 强制（不续费=停服） | 非强制（不续费=不升级，仍可用旧版） |
| 长期成本 | 不可控（年年涨） | 可控（买断+可选服务费） |

## 3. 定价策略与定价逻辑

### 3.1 定价原则

| 原则 | 说明 |
|------|------|
| **买断制为核心** | 顺应中小企业"一次付费、永久使用"的偏好，降低决策门槛 |
| **年度服务费非强制** | 不续费仍可用旧版，续费=升级+支持。避免 SaaS 的"绑架感" |
| **按站点数分档** | 单站点便宜，多站点贵，天然区分中小企业 vs 建站服务商 |
| **GEO 为溢价点** | GEO LLM 增强能力放在 Pro 版，作为差异化定价支撑 |
| **源码版高溢价** | 源码授权面向建站服务商/有开发能力的企业，高客单低销量 |
| **增值服务补充** | 托管/代建/代运营覆盖"买了不会用"的客户，提升 ARPU |

### 3.2 定价公式

```
License 买断价 = 研发分摊 + 交付成本 + 合理利润 + 版本差异溢价

年度服务费 = License 价 × 20%（约等于升级+支持的成本+利润）
```

### 3.3 心理定价设计

- Standard: **¥4,980**（"不到 5000 块，永久拥有"）
- Pro: **¥9,980**（"不到 1 万，多站点+GEO AI"）
- Source: **¥29,800**（"不到 3 万，源码+无限站"）
- 年度服务费：Standard ¥980/年、Pro ¥1,980/年、Source ¥3,980/年

> "不到 X" 的表述是刻意设计，让客户感觉"划算"。

### 5.3 License 状态与功能控制

```
License 验证流程：
  │
  ├─ License 有效 + 年度服务费未过期 → 全功能可用 + 可升级
  │
  ├─ License 有效 + 年度服务费过期 → 全功能可用 + 不可升级（锁定当前版本）
  │    └─ 后台提示"年度服务已过期，续费可获取新版本"
  │
  ├─ License 过期 → 管理功能锁定，已发布静态站保持访问
  │    └─ 后台显示"License 已过期，请联系续期"
  │
  └─ License 无效/篡改 → 同上 + 安全告警日志
```

> **关键设计**：年度服务费过期 ≠ 软件不可用。软件仍可正常运行当前版本，只是不能升级。这是与 SaaS 的核心差异，也是"买断制"承诺的兑现。

## 6. 增值服务设计

### 6.1 增值服务清单

| 服务 | 定价 | 适用客户 | 交付方式 | 毛利率 |
|------|------|---------|---------|--------|
| **托管部署** | ¥1,000/次 | 无 IT 人员的中小企业 | 远程协助，2 小时内完成 | ~80% |
| **代建站** | ¥2,000-5,000/站 | 有站但没时间/能力自建的企业 | 远程，1-3 天交付 | ~70% |
| **SEO/GEO 代运营** | ¥1,500/月起 | 建好站但不懂运营的企业 | 持续，月度报告 | ~75% |
| **模板定制** | ¥3,000-8,000/套 | 对设计有特殊要求的企业 | 远程，3-7 天 | ~65% |
| **培训服务** | ¥500/次 | 团队多人使用 | 线上 1 小时 | ~90% |

### 6.2 增值服务产品化逻辑

> 增值服务的核心价值：**覆盖"买了不会用"的客户群体**。

很多中小企业买了软件但不具备使用能力，增值服务把"软件交付"延伸到"效果交付"：

```
客户旅程与增值服务触点：

  购买 License
    │
    ├─ 不会部署？→ 托管部署（¥1,000）
    │
    ├─ 不会建站？→ 代建站（¥2,000-5,000）
    │
    ├─ 建好不会运营？→ SEO/GEO 代运营（¥1,500/月）
    │
    └─ 设计不满意？→ 模板定制（¥3,000-8,000）
```

### 6.3 SEO/GEO 代运营服务规格

| 月度交付物 | 内容 |
|-----------|------|
| 内容优化 | 每月优化 4-8 篇内容的 SEO/GEO（TDK + Schema + 可引用性提升） |
| 收录监控 | 监控百度/Google 收录状态，提交未收录页面 |
