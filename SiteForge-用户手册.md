## 1. 快速开始

### 1.1 首次登录

1. 打开浏览器访问系统地址（如 `http://localhost:3000`）
2. 输入管理员用户名和密码
3. 点击「登录」进入工作台

### 1.2 工作台概览

工作台是系统的核心入口，包含：

| 区域 | 内容 |
|------|------|
| 顶部统计卡 | SVI 搜索可见性指数、收录页面数、可引用性均分、近 7 天线索数 |
| 建站入口 | 参考同行建站 / 选择行业模板 / 从空白开始 |
| 待办清单 | SEO 未通过项、未处理线索、未配置 LLM API |
| 最近活动 | 最近的操作记录时间线 |

### 1.3 三种建站方式

| 方式 | 适合场景 | 操作 |
|------|---------|------|
| **参考同行** | 知道想要的网站结构 | 输入同行官网 URL → AI 解析结构 → 编辑内容 |
| **选模板** | 想快速套用行业模板 | 选行业 → 选模板 → 应用 → 编辑内容 |
| **从空白开始** | 有明确设计想法 | 直接进编辑器 → 拖拽组件搭建 |

## 2. AI 结构复刻（核心功能）

### 2.1 使用流程

1. 在工作台点击「参考同行」或侧边栏「建站 → AI 结构复刻」
2. **Step 1**：输入同行官网 URL（如 `https://www.example.com`）
3. 点击「解析并生成骨架」
4. **Step 2**：等待解析完成（5 个阶段：检查 robots.txt → 爬取渲染 → 识别页面类型 → 提取区块 → 生成模板）
5. **Step 3**：查看识别到的页面结构，勾选要保留的页面
6. 点击「确认进入编辑器」

### 2.2 合规须知

> ⚠️ 系统仅解析目标站的**页面结构**（HTML 标签层级、页面类型、区块位置），**不复制原文与图片**。生成骨架后所有文案由你重新填写。请确保目标站可公开访问，遵守其 robots.txt 与版权政策。

### 2.3 解析失败怎么办？

如果解析失败，系统会提示原因并提供「改用行业模板」的兜底入口。常见原因：
- 目标站需登录
- 目标站被反爬拦截
- 网络超时

## 3. 可视化编辑器

### 3.1 界面布局

编辑器采用三栏布局：

| 区域 | 功能 |
|------|------|
| 左栏：组件库 | 16 种标准组件（Hero/特性网格/产品网格/图文/FAQ/表单等），点击或拖入画布 |
| 中栏：画布 | 所见即所得预览，点击文字直接编辑，点击图片替换 |
| 右栏：属性面板 | 编辑选中区块的属性（文字内容/图片/链接/列数等） |

### 3.2 常用操作

| 操作 | 方法 |
|------|------|
| 添加区块 | 点击左栏组件 → 自动添加到画布 |
| 编辑文字 | 点击画布中的文字 → 直接输入 |
| 替换图片 | 点击图片 → 从媒体库选择或上传 |
| 调整顺序 | 拖拽区块上下移动 |
| 删除区块 | 选中区块 → 点击删除按钮 |
| 撤销/重做 | 顶栏撤销/重做按钮（≥ 20 步） |
| 切换页面 | 顶栏页面选择下拉框 |
| 预览 | 点击「预览」按钮，支持桌面/平板/手机三端 |
| 保存 | 点击「保存」按钮（自动保存每 30 秒） |

### 3.3 样式定制

在右栏属性面板底部：
- **主题色**：预设 5 套配色 + 自定义取色
- **字体**：预设字体组合

## 4. 内容管理（CMS）

### 4.1 内容类型

| 类型 | 用途 | 自定义字段示例 |
|------|------|--------------|
| 产品 | 产品展示页 | 规格、价格、参数 |
| 案例 | 成功案例页 | 客户、行业、挑战、方案、成果 |
| 文章 | 博客/新闻 | 作者、分类、标签 |

### 4.2 内容编辑

1. 侧边栏「内容 → 内容管理」
2. 点击「新建」或编辑已有内容
3. 填写标题、封面图、分类标签
4. 编辑正文（富文本编辑器）
5. **右侧 GEO 面板**实时显示可引用性评分和优化建议（详见第 7 节）
6. 底部 SEO 设置（标题/描述/URL slug）
7. 点击「保存」

### 4.3 页面与导航

- 侧边栏「内容 → 页面与导航」
- 左栏：页面树（可拖拽排序、≤ 2 级层级）
- 右栏：选中页面的属性（页面名、slug、SEO 配置）
- 底部：导航菜单配置（拖拽组织导航项）

## 5. 表单与线索管理

### 5.1 创建表单

1. 侧边栏「获客 → 表单设计器」
2. 左栏点击字段类型添加到表单（单行文本/电话/邮箱/多行文本等）
3. 配置字段属性（必填、提示文案、默认值）
4. 右栏配置投放：
   - **嵌入页面**：选择表单嵌入的页面
   - **弹窗触发**：开启后可选触发规则（滚动 50%/停留 30 秒/退出意图）
   - **浮动按钮**：页面右下角浮动咨询按钮
   - **通知配置**：邮件通知 + Webhook（微信/钉钉/飞书）

### 5.2 管理线索

1. 侧边栏「获客 → 线索管理」
2. 查看全部线索列表（支持按时间/来源/表单/状态筛选）
3. 点击「查看」进入线索详情
4. 切换跟进状态（未处理 → 已联系 → 已成交 / 无效）
5. 添加跟进记录
6. 点击「导出 CSV」批量导出

> 🔒 所有线索数据仅存储在本地服务器，不会外发。

## 6. SEO 引擎

### 6.1 SEO 设置

侧边栏「优化 → SEO 引擎」：

| 配置项 | 说明 |
|--------|------|
| 全局 SEO | 站点标题、描述、favicon、语言声明 |
| sitemap.xml | 自动生成，发布时更新 |
| robots.txt | 自定义规则，**默认放行 9 个 AI 爬虫**（GEO 关键） |
| Schema.org | 勾选要注入的结构化数据类型（Organization/Product/Article/FAQPage 等） |
| 搜索引擎收录 | 配置百度/Google/Bing 站长 API Key，一键提交 sitemap |
| 关键词监控 | 添加核心关键词，查看排名趋势 |

### 6.2 SEO 健康度检查

侧边栏「优化 → SEO 健康度」：

- 点击「重新检查」运行 12 项 SEO 检查
- 每项显示状态：✅ 通过 / ⚠️ 警告 / ❌ 未通过
- 支持自动修复的项目显示「一键修复」按钮
- 顶部显示 0-100 评分

### 6.3 AI 爬虫放行（GEO 关键）

> robots.txt 默认放行以下 AI 爬虫，这是 GEO 的关键操作：
> - GPTBot（ChatGPT）
> - ClaudeBot（Claude）
> - PerplexityBot（Perplexity）
> - CCBot（Common Crawl）
> - Google-Extended（Google AI）
> - OAI-SearchBot（ChatGPT Search）
> - Claude-SearchBot（Claude Search）
> - ChatGPT-User
> - Applebot-Extended（Apple Intelligence）

## 7. GEO 引擎（差异化核心）

GEO（生成式引擎优化）是 SiteForge 的核心差异化能力，面向 AI 搜索引擎（ChatGPT/Perplexity/文心一言/豆包/Kimi）。

### 7.1 llms.txt 编辑

侧边栏「优化 → GEO 引擎」：

1. 查看自动生成的 `llms.txt`（面向 AI 的站点说明文件）
2. 可手动编辑内容
3. 点击「重新生成」从站点数据重新生成
4. 发布时自动更新
5. 可切换查看 `llms-full.txt`（详细版）

### 7.2 可引用性评分

在「内容 → 编辑内容」页面的右侧 GEO 面板：

1. 编辑内容后点击「重新评分」
2. 系统从 6 个维度评分（总分 100）：

| 维度 | 满分 | 检测内容 |
|------|------|---------|
| 事实陈述密度 | 20 | 统计数据、百分比、数字出现频次 |
| 结构化程度 | 20 | FAQ、列表、表格、定义块占比 |
| 引用来源 | 15 | 外部链接、来源标注、专家引言 |
| 权威性信号 | 15 | 作者署名、头衔、发布时间、简介 |
| 内容完整度 | 15 | 字数、段落数、媒体数、独特性 |
| 语义清晰度 | 15 | H 标签层级、段落长度、关键词一致性 |

3. 根据优化提示改进内容
4. 重新评分验证提升效果

**评分等级：**
- 🟢 80-100：优秀，极易被 AI 引用
- 🟡 60-79：良好，可被引用
- 🟠 40-59：一般，需优化
- 🔴 0-39：较差，需重点优化

### 7.3 AI 引用模拟测试

侧边栏「优化 → AI 引用模拟」：

1. 输入测试问题（模拟用户在 AI 搜索中的提问）
2. 选择内容范围（全站/仅文章/仅产品）
3. 点击「运行测试」
4. 查看结果：
   - **引用概率**：🟢 高 / 🟡 中 / 🔴 低
   - **模拟回答**：AI 可能生成的回答
   - **命中片段**：检索到的相关内容
   - **优化建议**：如何提升引用概率
5. 配置 LLM API Key 后可获得更精准的模拟（详见第 10 节）

> ⚠️ 此为模拟测试，非真实 AI 搜索结果预测。实际效果受各平台算法影响。

### 7.4 GEO 写作模板

在内容编辑页右侧面板点击「应用 GEO 写作模板」：

| 模板 | 用途 |
|------|------|
| 行业问答（FAQ） | 生成常见问题结构，AI 搜索偏好 Q&A |
| 产品能力 | 表格化展示产品能力 + 数据指标 |
| 解决方案 | 行业挑战 → 方案 → 步骤 → 成果数据 |

## 10. LLM API 配置

### 10.1 为什么需要配置 LLM API？

GEO 引擎的 AI 引用模拟测试和智能写作建议需要调用大语言模型。未配置时使用规则引擎模式（基础功能可用），配置后解锁完整 AI 能力。

### 10.2 配置步骤

1. 侧边栏「系统 → LLM 配置」
2. 选择服务商：

| 服务商 | 推荐模型 | 特点 | 价格 |
|--------|---------|------|------|
| **DeepSeek**（推荐） | deepseek-chat | 性价比高 | ¥0.001/千 token |
| 阿里千问 | qwen-plus | 中文能力强 | ¥0.004/千 token |
| 智谱 GLM | glm-4-flash | 国内合规 | ¥0.001/千 token |
| OpenAI | gpt-4o-mini | 海外可选 | $0.15/百万 token |

3. 填入 API Key（在各服务商官网注册获取）
4. 点击「测试连接」验证
5. 点击「保存配置」

### 10.3 成本估算

| 操作 | 估算费用（DeepSeek） |
|------|---------------------|
| 单次 AI 引用模拟 | ≈ ¥0.001 |
| 单篇内容智能建议 | ≈ ¥0.001 |
| 月均使用成本 | 几元到十几元 |

### 10.4 合规说明

> ✅ 官网内容为公开信息，发送至 API 分析不涉及隐私。
> ✅ 线索/客户数据不会外发。
> ✅ 可随时清空 API Key 关闭 AI 功能，降级为规则引擎模式。

## 11. 常见问题

### Q: AI 解析失败怎么办？
A: 检查目标网站是否可公开访问、是否需登录。如仍失败，点击「改用行业模板」从模板库开始。

### Q: GEO 评分很低怎么提升？
A: 按右侧面板的优化提示逐项改进：补充统计数据（+33% 可见性）、添加 FAQ 结构、标注引用来源（+28%）、填写作者信息（+41%）。

### Q: 网站发布后搜不到？
A: 1) 检查 SEO 健康度是否通过；2) 在 SEO 设置中提交 sitemap 到百度/Google；3) 确认 robots.txt 未屏蔽搜索引擎。

### Q: AI 引用模拟显示"不太可能被引用"？
A: 说明内容质量不足以被 AI 搜索引用。按优化建议：补充数据、增加 FAQ、标注作者和来源，然后重新测试。

### Q: License 过期了网站还能访问吗？
A: 已发布的静态网站保持正常访问。仅管理/编辑功能锁定，续费后恢复。

### Q: 线索数据存在哪里？
A: 所有线索数据 100% 存储在你自己的本地服务器，不会外发。

### Q: 如何切换明暗主题？
A: 点击顶栏右上角的太阳/月亮图标切换主题，系统会记住你的偏好。

### Q: 如何在手机上预览网站？
A: 在编辑器中点击预览，切换到手机视口（375px 宽度）查看移动端效果。
